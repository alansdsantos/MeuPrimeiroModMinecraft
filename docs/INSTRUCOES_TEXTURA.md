# Instruções para Criar a Textura do Item

A textura atual (`meu_primeiro_item.png`) é apenas um placeholder. Para criar sua própria textura:

## Opções:

### 1. Usar um Editor de Imagens
- Use qualquer editor de imagens (Paint, GIMP, Photoshop, etc.)
- Crie uma imagem de **16x16 pixels** ou **32x32 pixels**
- Salve como PNG na pasta: `src/main/resources/assets/meuprimeiromod/textures/item/meu_primeiro_item.png`

### 2. Usar Ferramentas Online
- [Piskel](https://www.piskelapp.com/) - Editor de sprites online
- [Aseprite](https://www.aseprite.org/) - Editor profissional (pago, mas tem trial)

### 3. Usar Texturas do Minecraft
- Você pode copiar uma textura existente do Minecraft como base
- As texturas do Minecraft estão em: `.minecraft/versions/[versão]/[versão].jar`
- Extraia o JAR e procure em `assets/minecraft/textures/item/`

## Dicas:
- Use transparência (fundo transparente) para melhor integração
- Mantenha o estilo pixel art consistente com o Minecraft
- Teste a textura no jogo para ver como fica
